Kitten Invaders game

A game I developed for Windows PC.

This game is still under development. It is my final project for a Python course I've been attending. The game is developed with Python/pygame.

Instructions:

  1. Download KittenInvaders alt. a or b:
    1. kitteninvaders.zip (--onedir version, hopefully ok for Win10 Defender?)
    2. kitteninvaders_x.zip (--onefile version, flagged by Win10 Defender as threat)
  2. Extract files to your computer (right-click zip-file and click "Extract all")
  3. In the extracted folder, double-click kitteninvaders.exe

(Apparently Windows Defender (on Win10) and some antivirus software flag exe-files built with pyinstaller as threats. If I figure out how to build them in a way that they don't cause an alert, I will certainly update the downloads here.)

If you prefer, you can download the code from my github repository, and install the necessary environment (python and pygame, see requirements.txt) on your PC to run it.
